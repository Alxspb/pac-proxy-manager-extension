지리적으로 제한된 콘텐츠 접근, 연결 보안, 또는 기업 네트워크 요구 사항을 쉽게 관리하는 데 완벽합니다.

✨ 주요 기능:
• 지능형 PAC 스크립트 관리 - URL에서 로드하거나 직접 붙여넣기
• 와일드카드 지원이 있는 도메인별 예외 (*.example.com)
• 여러 프록시 서버 구성 (HTTP, HTTPS, SOCKS4, SOCKS5)
• 기업 설정을 위한 대량 도메인 가져오기
• 지능형 폴백이 있는 실시간 프록시 전환
• React로 구축된 현대적이고 반응형 인터페이스
• 다국어 지원 (13+개 언어)

🔒 개인정보보호 및 보안:
• 100% 무료 및 오픈 소스 - 코드를 직접 검사해보세요
• 최소한의 권한 (3개만 필요) - 최대 개인정보보호
• 모든 데이터는 귀하의 기기에 로컬 저장
• 추적 없음, 분석 없음, 데이터 수집 없음
• 보안 강화를 위해 Manifest V3로 구축

🚀 작동 방식:
1. 확장 프로그램을 설치하고 툴바 아이콘을 클릭
2. URL에서 PAC 스크립트를 추가하거나 스크립트 내용을 붙여넣기
3. 세밀한 제어를 위해 도메인 예외 구성
4. 필요한 경우 프록시 서버 설정
5. 확장 프로그램이 자동으로 프록시 라우팅 처리

💡 다음과 같은 용도에 완벽:
• 지리적으로 제한된 콘텐츠 및 스트리밍 서비스 접근
• 공공 WiFi 네트워크에서 연결 보안
• 기업 리소스에 연결하는 원격 근무자
• 학술 콘텐츠에 접근하는 학생 및 연구자
• 여러 네트워크 요구 사항을 관리하는 비즈니스 전문가
• 수동 번거로움 없이 자동 프록시 전환을 원하는 모든 사람

⚙️ 권한 설명:
최대 투명성을 위해 최소한의 3개 권한만 사용:
• "proxy" - 브라우저의 프록시 설정을 수정하는 데 필요
• "activeTab" - 쉬운 예외 설정을 위해 현재 탭의 도메인을 감지하는 데 사용
• "storage" - 귀하의 기기에 구성을 로컬로 저장

🎯 지능형 우선순위 시스템:
1. 도메인 예외가 최고 우선순위
2. 구성된 프록시 서버가 PAC 스크립트 프록시를 재정의
3. 구성된 프록시 서버가 없을 때 PAC 스크립트 규칙 적용
4. 대체 수단으로 직접 연결

🌐 기업 준비:
• 텍스트 목록에서 예외 도메인 대량 가져오기
• 복잡한 PAC 스크립트 논리 지원
• 여러 프록시 서버 구성
• 쉬운 예외 관리를 위한 와일드카드 도메인 매칭

🔧 고급 기능:
• URL에서 PAC 스크립트 실시간 재로드
• 깔끔하고 직관적인 탭 인터페이스
• 실시간 구성 업데이트

PAC Proxy Manager를 선택하는 이유?
폐쇄 소스 블랙박스인 다른 프록시 확장 프로그램과 달리, 저희는 완전히 투명합니다. 모든 코드 라인이 GitHub에서 이용 가능합니다. 저희의 최소 권한 모델은 귀하가 방문하는 웹사이트를 저희가 모른다는 것을 의미합니다 - PAC 스크립트는 Chrome 자체에서 관리하는 통제되고 캡슐화된 환경에서 실행됩니다.

다른 프록시 관리자들은 종종 "모든 웹사이트 읽기" 또는 "모든 데이터 수정"과 같은 광범위한 권한을 요구합니다. 저희는 필요한 것만 정확히 요청하고 그 이상은 요구하지 않습니다.
